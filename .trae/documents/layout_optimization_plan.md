# 策略对比页面全面布局优化计划

| 实时状态     | 水平排列         | 紧凑，可优化布局层次           |

## 优化方案

### 1. 页面整体结构优化

**目标**：提升页面整体层次感和视觉引导

**修改内容**：

- 添加页面顶部装饰条（渐变背景）
- 优化容器最大宽度（从1400px调整为1600px）
- 增加页面主副标题结构
- 添加页面背景网格效果

### 2. 头部区域优化

**目标**：增强品牌识别和信息展示

**修改内容**：

- 标题增加图标和副标题
- 实时状态区域重新布局（左对齐）
- 时间戳和刷新按钮样式优化
- 添加数据更新频率提示

### 3. 图例区域优化

**目标**：提升图例交互性和视觉吸引力

**修改内容**：

- 图例项增加点击交互（可筛选策略）
- 添加策略描述文字
- 悬停时显示策略颜色预览
- 增加图例边框动画

### 4. 摘要卡片优化

**目标**：提升数据展示清晰度和视觉冲击力

**修改内容**：

- 卡片增加背景渐变效果
- 统计项增加图标
- 平均收益增加趋势指示（上升/下降箭头）
- 悬停效果增强

### 5. 日期标签优化

**目标**：提升日期选择体验

**修改内容**：

- 日期标签增加选中状态动画
- 添加日期范围导航（上一周/下一周）
- 周末日期增加斜线装饰
- 增加今日标记

### 6. 策略对比区域优化

**目标**：提升股票列表可读性和交互体验

**修改内容**：

- 策略列增加滚动指示器
- 股票卡片增加展开动画
- 股票详情表格增加横向滚动支持
- 添加股票搜索功能

### 7. 共同推荐区域优化

**目标**：突出共同推荐股票的价值

**修改内容**：

- 标题增加装饰元素
- 股票卡片增加策略标签展示
- 添加收益对比信息
- 增加悬停动画效果

### 8. 响应式布局优化

**目标**：确保移动端完美适配

**修改内容**：

- 优化平板端布局（768px-1024px）
- 手机端表格横向滚动
- 触控友好的按钮尺寸
- 字体大小自适应

## 潜在依赖与风险

### 依赖

- 无需新增外部依赖
- 使用现有 CSS/JavaScript 实现

### 风险

- 布局调整可能影响现有功能
- 响应式修改可能引入新的兼容性问题
- 交互增强可能增加代码复杂度

### 风险应对

- 每阶段修改后进行测试验证
- 保留原有功能的兼容性
- 使用渐进增强方式实现新功能
